# ElmerStudio tutorials

Project files for the ElmerStudio video tutorial series.
Each subfolder contains the `.sif`, `.grd`, diagrams, and a
README with reproduction steps.

| #  | Topic                                  | Video |
|----|----------------------------------------|-------|
| 01 | [Heat conduction on a 2D plate](./01-heat-conduction-2d/)          | [YouTube](https://youtu.be/DGWUpYdniXY) |
| 02 | Anatomy of a SIF file                  | (planned) |
| 03 | Mesh generation from scratch           | (planned) |
| 04 | Cantilever beam: linear elasticity     | (planned)    |
| 05 | Parametric sweeps                      | (planned)    |
| 06 | Stress concentration (Bodies & Boundaries) | (planned) |
| 07 | Importing external meshes (Gmsh, etc.) | (planned)    |

## Requirements

- ElmerStudio (any 0.7+ release from this repository)
- Elmer FEM (the solver itself), from <http://www.elmerfem.org/>

See each subfolder's README for problem-specific notes.

## Contributions

Typos and clarifications are welcome via pull request. For substantive
changes — different problems, different physics — please open an issue
first so we can talk through scope.
